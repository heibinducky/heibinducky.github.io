About Us

Hi, I'm illnacio and I'm a 24 year old Chilean student who's really passionate about drawing.

I'm currently working on a webcomic project which I'm doing completely own. I plan to release this series in both Spanish and English doing all the work by myself. So I end up coming up with the story, making the comic itself, making the scripts and then translating. The website is also done by me.

So yeah I'm really putting all my efforts into this, and if you stumble upon this page and want to contribute I'd really appreciate it!

If there's any type of contribution I'd make sure to thank you in the next volume of my comic alongside other places. And you can also get some extra content in a special release as well.

The comic's first ever volume was released on March 18th, and the second one is set to release on April 15th. You can read more about it on my website illnacio.com. The comic is completely free.

Thank you for reading, have a great day!
